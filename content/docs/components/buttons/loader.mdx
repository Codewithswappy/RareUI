## Features

- **Geometric Animation**: Multiple rotating polygons create a mesmerizing effect
- **SVG Masking**: Uses SVG masks for clean, scalable animations
- **Smooth Rotation**: Staggered animation delays for fluid motion
- **Green Theme**: Built with green-500 color, easily customizable
- **Lightweight**: Pure CSS and SVG, no dependencies

## Customization

### Change Color

Replace the green color with your own:

```tsx
<div style={{ background: '#your-color' }} />
```

### Adjust Size

Modify the `transform: scale()` value:

```tsx
<div style={{ transform: 'scale(0.75)' }}> {/* Larger */}
```

### Animation Speed

Change the animation duration in the polygon classes:

```tsx
animate-[rotation_1s_linear_infinite] {/* Faster */}
```
