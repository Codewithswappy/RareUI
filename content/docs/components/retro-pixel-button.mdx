## Usage

```tsx
import RetroPixelButton from '@/components/rareui/retro-pixel-button';

export default function App() {
  return (
    <RetroPixelButton onClick={() => alert('Clicked!')}>
      CLICK ME
    </RetroPixelButton>
  );
}
```

## Examples

### Custom Colors

```tsx
<RetroPixelButton 
  pixelColor="#00ff00"
  baseColor="#1a1a1a"
  textColor="#ffffff"

</RetroPixelButton>
```

### Different Sizes

```tsx

  LARGE BUTTON
</RetroPixelButton>
```

## Features

- **Retro Pixel Aesthetic**: Font-mono styling with pixelated arrow icon
- **Sliding Animation**: Smooth spring-based expansion of the pixel block on hover
- **Icon Rotation**: Pixelated arrow rotates 180Â° during hover state
- **Text Morphing**: Text subtly blurs and shifts during transition for a glitch-like effect
- **Customizable Colors**: Control pixel block, base, and text colors independently
- **Tap Feedback**: Subtle scale-down effect on click for tactile response
- **Theme Support**: Works seamlessly with light and dark modes
- **Framer Motion**: Powered by smooth, physics-based animations
- **Fully Type-Safe**: Complete TypeScript support with proper prop types

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `children` | `React.ReactNode` | `"TRY FRAMER FOR FREE"` | The button text content. |
| `className` | `string` | `-` | Additional CSS classes for the button. |
| `pixelColor` | `string` | `"orange"` | Color of the sliding pixel block (CSS color value). |
| `baseColor` | `string` | `-` | Optional background color override for the button. |
| `textColor` | `string` | `-` | Optional text color override. |
| `...props` | `React.ButtonHTMLAttributes<HTMLButtonElement>` | `-` | All standard button HTML attributes (onClick, disabled, etc.). |
